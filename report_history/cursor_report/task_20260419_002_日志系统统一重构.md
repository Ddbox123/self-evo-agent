## 2. 设计

**核心原则：**
- 单一日志来源：所有模块统一使用 `from core.logging import debug_logger`
- 渐进式迁移：保留 `import logging` 仅用于 `setup_logging()` 配置
- 条件导入：避免循环依赖，优先使用函数内导入

**统一日志接口：**
```python
# 方式一：直接使用 debug_logger
from core.logging import debug_logger
debug_logger.info("消息")
debug_logger.warning("警告")
debug_logger.error("错误")
debug_logger.debug("调试")

# 方式二：通过 logger 别名
from core.logging import logger
logger.info("消息")
```
