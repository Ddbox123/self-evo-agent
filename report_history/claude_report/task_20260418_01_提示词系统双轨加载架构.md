## 架构设计

### 双轨加载策略

```
core/core_prompt/              ← 静态核心提示词（内置只读模板）
workspace/prompts/            ← 动态提示词（用户可编辑，覆盖优先）

加载优先级：
1. SOUL.md / AGENTS.md：优先 workspace，回退 static
2. IDENTITY.md / USER.md / DYNAMIC.md / COMPRESS_SUMMARY.md：仅 workspace
```

### CorePromptManager 核心方法

| 方法 | 策略 |
|------|------|
| `load_soul()` | workspace 优先，回退 static |
| `load_agents()` | workspace 优先，回退 static |
| `load_identity()` | 仅 workspace |
| `load_user()` | 仅 workspace |
| `load_dynamic()` | 仅 workspace |
| `load_compress_summary()` | 仅 workspace |

## 遇到问题

### 问题1：workspace_manager.py 路径错误

**描述**：`workspace_manager.py` 位于 `core/infrastructure/`，使用 `parent.parent` 只能退到 `core/`，但 workspace 在项目根目录。

**解决方案**：改为 `parent.parent.parent`，回退到 `sys.path` 搜索 `agent.py`。

### 问题2：INDEX.md 编辑器模糊匹配失败

**描述**：StrReplace 工具的模糊匹配导致编辑失败。

**解决方案**：使用 Python 脚本直接替换文本内容。
