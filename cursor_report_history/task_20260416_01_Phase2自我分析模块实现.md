## 能力维度评估

实现后新增 10 个能力维度的评分算法：

| 维度 | 说明 |
|------|------|
| `code_quality` | 代码质量 - 规范、文档、类型注解 |
| `test_coverage` | 测试覆盖 - 文件数、覆盖率 |
| `tool_usage` | 工具使用 - 数量、分类、成功率 |
| `autonomy` | 自主性 - 状态管理、事件驱动 |
| `memory_management` | 记忆管理 - 工具、工作区、世代 |
| `security` | 安全性 - 模块、沙箱、白名单 |
| `architecture` | 架构设计 - 模块化、代码精简 |
| `learning` | 学习能力 - 洞察、历史积累 |
| `planning` | 规划能力 - 任务规划、优先级 |
| `execution` | 执行能力 - 执行器、超时控制 |

## 遇到的问题

1. **pytest-asyncio 未安装** - 异步测试无法运行
   - 解决：执行 `pip install pytest-asyncio`

2. **中文字符编码问题** - PowerShell 显示乱码
   - 解决：使用 UTF-8 编码读取文件

3. **浮点数精度问题** - 测试断言失败
   - 解决：使用 `abs(a - b) < 0.001` 而非 `a == b`

4. **模块优先级匹配问题** - "test" 匹配了 "tools"
   - 解决：调整条件判断顺序，更具体匹配在前
