## 任务概述

**任务名称：** ASCII 形象选择系统

**目标：** 实现用户可根据预设图案选择自己想要的形象功能

**背景：** 用户反馈当前 ASCII 形象不太像龙虾宝宝，希望提供多种可爱形象供选择

## 使用方式

### 配置文件
```toml
[avatar]
preset = "shrimp"
```

### 命令行切换
```
/avatar          # 查看所有可用形象
/avatar shrimp   # 切换到小虾米
```
