# Decision Module (Phase 6)

**自主决策模块** - 决策树与策略选择

## Modules

| File | Description |
|------|-------------|
| `decision_tree.py` | 决策树实现 |
| `priority_optimizer.py` | 优先级优化器 |
| `strategy_selector.py` | 策略选择器 |

## Usage

```python
from core.decision.decision_tree import DecisionTree
from core.decision.priority_optimizer import PriorityOptimizer
from core.decision.strategy_selector import StrategySelector
```

## Key Classes

- `DecisionTree` - 基于规则的决策树
- `PriorityOptimizer` - 工具调用优先级优化
- `StrategySelector` - 执行策略自动选择

## 功能

- 条件分支决策
- 工具调用优先级排序
- 自适应策略选择
