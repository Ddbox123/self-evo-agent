# 🦞 虾宝宠物系统 - 创意设计报告

> **版本：** v3.0 | **日期：** 2026-04-19 | 
> **状态：** 🔥 核心功能已完成 | **特色：** Token 喂养 + AI 情感生成

---

## 🎯 系统愿景

**「不只是宠物，是数字生命的觉醒」**

虾宝宠物系统颠覆传统 AI Agent 的冰冷形象，将每一次 Token 交互转化为"生命成长"的养成分体验。

> 传统的 AI 对话消耗的是"tokens"
>
> 虾宝系统让每一次 tokens 消耗都成为"喂养"你的数字伙伴的营养

---

## 🌟 核心创意亮点

### 1. 🍖 Token 喂养机制

**「你的 AI 宠物靠 tokens 长大」**

```
┌─────────────────────────────────────────────────────────────────┐
│                     Token 喂养循环                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   ┌──────────┐         tokens              ┌──────────┐          │
│   │   你    │ ──────────────────────────▶ │   虾宝   │          │
│   │  (输入)  │                             │   宠物   │          │
│   └──────────┘                             └──────────┘          │
│                                                  │               │
│                                                  ▼               │
│                                          ┌──────────────┐       │
│                                          │   饭量计算    │       │
│                                          │ context_window│       │
│                                          └──────────────┘       │
│                                                  │               │
│                                                  ▼               │
│                                          ┌──────────────┐       │
│                                          │   饱食度     │       │
│                                          │  hunger/100   │       │
│                                          └──────────────┘       │
│                                                  │               │
│                                                  ▼               │
│                                          ┌──────────────┐       │
│                                          │   心情值     │       │
│                                          │  mood/100   │       │
│                                          └──────────────┘       │
└─────────────────────────────────────────────────────────────────┘
```

**设计原理：**
- **饭量** = 上下文窗口 × 0.1
- **饱食度** = 已消耗 tokens / 饭量
- **心情系统** = 饱食度 + 任务完成度 + 模型互动质量

## 🎮 宠物属性详解

### 基础属性

| 属性 | 范围 | 说明 | 影响因素 |
|------|------|------|---------|
| **心情 (mood)** | 0-100 | 宠物情绪状态 | 对话内容 + 饱食度 + 任务完成 |
| **饱食度 (hunger)** | 0-100 | 饥饿程度 | Token 消耗速度 |
| **活力 (energy)** | 0-100 | 行动能力 | 休息恢复 / 任务消耗 |
| **健康 (health)** | 0-100 | 身体状态 | 响应时间 + 错误率 |
| **爱心 (love)** | 0-100 | 与主人亲密度 | 夸奖次数 + 互动质量 |
| **等级 (level)** | 1-∞ | 成长阶段 | 经验值累积 |

### 经验值系统

```
升级公式: exp_to_next = 100 × 1.5^(level-1)

Lv.1 → Lv.2: 100 exp
Lv.2 → Lv.3: 150 exp
Lv.3 → Lv.4: 225 exp
...
Lv.9 → Lv.10: 满级，触发特殊成就！
```

### 子系统职责

| 系统 | 职责 | 关键方法 |
|------|------|---------|
| **GeneSystem** | 基因遗传、模型特征 | `update_context_window()` |
| **HeartSystem** | 心跳动画、生命体征 | `beat()`, `get_animation()` |
| **DreamSystem** | 梦境生成、记忆压缩 | `enter_dream()`, `get_fragment()` |
| **PersonalitySystem** | 性格养成、行为记录 | `record_behavior()`, `get_dominant_trait()` |
| **HungerSystem** | Token 喂养、饭量计算 | `record_tokens()`, `_feed()` |
| **DiarySystem** | 成长日记、情感分析 | `add_entry()`, `_analyze_sentiment()` |
| **HealthSystem** | 健康体检、指标监控 | `_update_metrics()` |
| **SocialSystem** | 模型社交、好友管理 | `meet_model()`, `interact()` |
| **SkinSystem** | 成就解锁、皮肤管理 | `check_achievement()`, `unlock_skin()` |
| **SoundSystem** | 情绪声音、动作音效 | `play_sound()` |
